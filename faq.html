Installation

It's available on iOS, Android and on the web . It is recommended that you install the mobile app version on your smartphone.

The iOS and Android versions are found on the Apple App Store and Google Play Store, respectively. Search for "Emora Chat" and make sure that the developer is listed as Emory University.

Yes, the web version is accessed via your browser and found here . We recommend using chrome for the web version.

Use your Emory NetID and password. The NetID should not include "@emory.edu".

Account

Yes, you need a valid Emory account called a NetID to use Emora Chat. There is no need to create a separate account!

No, a valid Emory NetID and password are required to log in; without one, you can not log in. If they are an Emory student, they can use their personal NetID and password to log in.

Absolutely not. Sharing your NetID and password is a violation of Emory policies.

No, accepting the TOS fully is required to use Emora Chat.

After having logged in, in the top left corner is a hamburger menu (three dashes); the terms of service can be viewed by clicking on this menu and clicking on "Terms Of Service" in the pop-up.

If you are being asked to accept the TOS again, this means that it was updated. Every time the TOS is updated, users must accept it again.

Security

Yes, while the login process does not take you to the main Emory login page (like how logging into OPUS would), it is secure and safe to use because this application was developed by the Computer Science department at Emory.

No, a VPN is not required to use Emora Chat.

No, being connected to Emory's wifi is not required to utilize Emora Chat. However, an internet connection is required to communicate with Emora Chat; this can be EmoryUnplugged or EmoryGuest, the cellular network that your smartphone uses, or other wireless networks.

Functionality

Your messages are located on the right side of the screen on a purple chat bubble, the bot's responses are located on the left side of the screen with a gray chat bubble. The bot's responses also have an icon to the right of them for feedback.

Emora Chat is currently a self contained experience. It has set conversation states; outside of these, it is incapable of responding. If there is no response at all from the bot, there is likely a server side issue. After a few minutes, functionality should be restored.

This icon is a way to give feedback on the responses Emora Chat gives. This is useful for the researchers that created the bots to understand what works well, and what does not!

This button is intended to allow users to send a file/picture to Emora; however, at this time, there is no functionality requiring users to send attachments. Emora does not respond to attachments sent.

A server error message is indicative of the server being offline; this could be due to an update that is being pushed out, a bug in the server/app, connectivity issue, etc. Check back again soon, and it should return to normal.

You are likely not connected to the internet. Emora Chat requires an internet connection to function, please connect to the internet to utilize Emora. If you are connected, and the issue persists, please report the problem. The best way to report the problem is to send email to emorachat@emory.edu.

Miscellaneous

Emora Chat itself is not open source; however, it is built upon Flutter, a framework created by Google LLC that is open source. Flutter uses the Dart programming language, which is also open source. Emora Chat incorporates various packages developed by Emory NLP and third parties, and their respective licenses can be found in the app's 'About' screen.

The minimum supported iOS version is 11.0 or newer, and the minimum supported Android version is 4.4 (SDK version 19) or newer. Please ensure your device runs iOS 11.0 or later or Android 4.4 or later to use Emora Chat.

The best way to request a new feature is to send email to emorachat@emory.edu
