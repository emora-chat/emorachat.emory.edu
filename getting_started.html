Getting Started

Installation

Emora Chat is accessible via a mobile app (recommended) or a web browser. The mobile apps can be installed on both iOS and Android devices via the App Store and Google Play, respectively:



We recommend using Google Chrome for the web interface, which provides a convenient way to experience Emora Chat. However, due to the diversity of web browsers, it may have limitations in supporting all features. In contrast, the mobile apps provide a comprehensive set of features and undergo more rigorous testing than the web interface. Therefore, we strongly recommend installing the mobile app on your device to fully enjoy the Emora Chat experience.

Login

When you open the Emora Chat app or access the web interface after installation, you will be greeted with the following login page:

Enter your Emory Net ID and password, then click the  Login  button. Currently, Emora Chat only supports users with Emory credentials, which are securely transmitted through the Lightweight Directory Access Protocol (LDAP) authentication.

Dashboard

Once you log in, you will see the following dashboard:

When you click the menu button on the top left corner ( ☰ ), it deplays the following options:

  • About
  • FAQ
  • Terms of Service

Below Emora's greeting, you will find the available chatbots for you to interact with:

  • Information Bot
  • Assistant Bot
  • Companion Bot

Select any of the chatbots to initiate your first conversation. If you click the logout button on the top right corner ( [➡ ), it prompts you to confirm if you want to sign out.

Chat Interface

Our chat interface is similar to the standard layout used in other messaging apps. Type your message in the text box at the bottom and send it by clicking the send button on the right (➤). The chatbot response will be displayed below your message, which may take 2-5 seconds:

You may have noticed that there are two buttons on the right-hand side of the chatbot response. The first box (in pink) is for providing feedback about this specific chatbot, which may not appear for all chatbots. Once you have submitted feedback by selecting one of the options, the button's color will change to light gray, as illustrated in the right figure below. You can modify your feedback by clicking the button again:

The second box (in light gray) is for providing general feedback, offering three options: thumbs up (👍), thumbs down (👎), and cancel (🧼). Once you select an option, the button will change to the chosen option, as illustrated in the right figure below. You can modify your feedback by clicking the button again:
